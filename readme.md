# wsen

> World of Warcraft Server Emulation in node.js

Currently supports the `0.5.3` (3368 build) alpha client version.

## Features

- Authentication
- Character enumeration
- Character creation
- World login

## Install

```sh
npm i
```

## Usage

```sh
node index
```
